# ساعتي - Sa3aty

## نبذة عن المشروع

**ساعتي** هو تطبيق ويب لتتبع الوقت بطريقة سهلة وبسيطة. يساعدك على تسجيل ساعات العمل على المشاريع المختلفة مع دعم كامل للغة العربية والاتجاه من اليمين إلى اليسار (RTL).

## المميزات

- ✅ **تصميم متجاوب**: تصميم سطح مكتب مع شريط جانبي وتصميم متوافق مع الأجهزة المحمولة
- ✅ **تطبيق ويب تقدمي (PWA)**: يعمل بدون اتصال بالإنترنت
- ✅ **أصوات محيطة**: مشغل صوت المطر المدمج للتركيز والاسترخاء
- ✅ **تتبع الوقت**: مؤقت سهل الاستخدام مع إدارة المشاريع
- ✅ **دعم RTL**: دعم كامل للغة العربية مع تخطيط من اليمين إلى اليسار
- ✅ **نشر تلقائي**: نشر تلقائي على GitHub Pages مع كل تحديث

## كيفية تشغيل المشروع

### المتطلبات الأساسية

قبل البدء، تأكد من تثبيت:
- **Node.js** (الإصدار 18 أو أحدث) - [تنزيل Node.js](https://nodejs.org/)
- **npm** (يأتي مع Node.js) أو **yarn** أو **bun**

يمكنك التحقق من تثبيت Node.js عبر تشغيل:
```sh
node --version
npm --version
```

### خطوات التثبيت والتشغيل

#### 1. استنساخ المستودع
```sh
git clone https://github.com/Ahmad-Alqattu/sa3aty-time-tracker.git
```

#### 2. الانتقال إلى مجلد المشروع
```sh
cd sa3aty-time-tracker
```

#### 3. تثبيت المكتبات والتبعيات
```sh
npm install
```

أو إذا كنت تستخدم yarn:
```sh
yarn install
```

أو إذا كنت تستخدم bun:
```sh
bun install
```

#### 4. تشغيل المشروع في وضع التطوير
```sh
npm run dev
```

أو:
```sh
yarn dev
```

أو:
```sh
bun dev
```

بعد تشغيل الأمر، افتح المتصفح وانتقل إلى:
```
http://localhost:5173
```

### أوامر إضافية مفيدة

#### بناء المشروع للإنتاج
```sh
npm run build
```

#### معاينة النسخة المبنية
```sh
npm run preview
```

#### فحص الكود (Linting)
```sh
npm run lint
```

#### تشغيل الاختبارات
```sh
npm run test
```

#### تشغيل الاختبارات في وضع المراقبة
```sh
npm run test:watch
```

## التقنيات المستخدمة

هذا المشروع مبني باستخدام:

- **Vite** - أداة بناء سريعة
- **TypeScript** - لغة برمجة مكتوبة
- **React** - مكتبة واجهة المستخدم
- **shadcn/ui** - مكتبة مكونات واجهة المستخدم
- **Tailwind CSS** - إطار عمل CSS
- **React Router** - للتنقل بين الصفحات
- **Zustand** - لإدارة الحالة (State Management)

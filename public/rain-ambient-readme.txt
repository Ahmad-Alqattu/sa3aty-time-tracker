# Ambient Audio Files

This directory should contain three ambient sound files for the Sa3aty Time Tracker application:

## Required Audio Files

### 1. Rain Sound (مطر)
- **Filename**: `rain-07.mp3`
- **Source**: https://www.soundjay.com/nature/rain-07.mp3
- **Description**: Relaxing rain sound for focus

### 2. Cafe Ambience (مقهى)
- **Filename**: `cafe-ambience.mp3`
- **Source**: https://upload.wikimedia.org/wikipedia/commons/a/ab/Ambience_of_a_busy_cafe.mp3
- **Description**: Busy cafe atmosphere sound

### 3. Brown Noise (ضجيج بني)
- **Filename**: `brown-noise.mp3`
- **Source**: https://cdn.pixabay.com/download/audio/2022/02/03/audio_5131105e4c.mp3
- **Description**: Brown noise for concentration

## How to Add Audio Files

**Option 1: Download directly from sources**
Download each file from the URLs above and place them in this `public` directory with the exact filenames listed.

**Option 2: Use your own audio files**
You can use any MP3 files you prefer, but make sure to:
- Use the exact filenames listed above
- Ensure files are in MP3 format
- Make sure the audio loops smoothly (no harsh transitions at start/end)
- Recommended duration: 30 seconds or longer

## File Specifications

- **Format**: MP3
- **Loopable**: Yes (seamless loop recommended)
- **Quality**: Good quality for ambient background
- **Duration**: 30+ seconds recommended

## Fallback

If local files are not available, the app will attempt to load from the CDN sources as a fallback.
